## 📋 Sobre o projeto
<p>Esta aplicação web foi desenvolvida para facilitar a organização de sorteios de Amigo Secreto.</p>
<p>O sistema permite cadastrar participantes, validar entradas e realizar um sorteio aleatório garantindo que todos os participantes sejam incluídos em um ciclo de troca de presentes.</p>


## ✦ Principais funcionalidades:

<p>➥ Cadastro de Participantes: Adicione nomes à lista de sorteio através de um campo de entrada intuitivo.</p>
<p>➥ Validações Inteligentes: <br>
    •Impede a adição de campos vazios.  <br> 
    •Evita a duplicidade de nomes na lista.   <br> 
    •Exige um mínimo de 4 participantes para garantir um sorteio viável.</p>
<p>➥ Algoritmo de Sorteio: Utiliza uma lógica de embaralhamento aleatório (Fisher-Yates) para garantir resultados imparciais.</p>
<p>➥ Exibição de Resultados: Apresenta os pares formados (ex: Amigo A -> Amigo B) diretamente na tela.</p>
<p>➥ Reinicialização: Função para limpar todas as listas e reiniciar o processo do zero.</p>
